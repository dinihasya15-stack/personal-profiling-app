# Personal Profile Web Application

A simple Java web application for creating student profiles using Servlets and JSP.

## Features
- Student profile form with validation
- Profile display page
- Session management
- Responsive design with FSKM background

## Technologies Used
- Java Servlets
- JSP (JavaServer Pages)
- HTML5 & CSS3
- JavaScript
- Apache Tomcat/GlassFish Server

## Project Structure


## Setup Instructions

### Prerequisites
- Java JDK 8 or higher
- Apache NetBeans IDE
- Apache Tomcat 9.x or GlassFish Server

### Installation Steps
1. Clone this repository
2. Open in NetBeans IDE
3. Configure your web server (Tomcat/GlassFish)
4. Deploy and run the application

### How to Use
1. Access `index.html` to fill the student profile form
2. Submit the form to create a profile
3. View the generated profile on `profile.jsp`
